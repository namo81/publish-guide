Float

Float 기본

block 타입의 가로 배열 시 사용 (div, p 등)

가로배열을 감싸는 태그에 'float-wrap' 사용

float-wrap 는 overflow:hidden 으로 설정되어 있음 - float 영역 이후로 이어지는 컨텐츠가 float 의 영향을 받지 않도록.

fl = float:left / fr = float:right 속석 적용

fl, fr 태그에 별도의 width 값이 없을 경우 포함하고 있는 컨텐츠 너비로 계산됨.

Left Content
Right Content
<div class="float-wrap">
	<div class="fl">Left Content</div>
	<div class="fr">Right Content</div>
</div>

	<div class="fr" style="width:40%;">Right Content</div>
</div>

Clear 방식

float-wrap 일 경우 해당 영역 내 컨텐츠가 영역을 벗어나서 보여야 할 경우 overflow:hidden 으로 인해 보이지 않음.

동일한 효과를 유지하면서 영역을 벗어난 컨텐츠가 보여야 할 경우 사용.

이 경우 float-clear 영역 내 :after 요소가 추가됨.

Left Content
Right Content
<div class="float-clear">
	<div class="fl">Left Content</div>
	<div class="fr">Right Content</div>
</div>
