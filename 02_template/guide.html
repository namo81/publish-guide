말풍선

버튼(btn-balloon) + 말풍선 컨텐츠(cnt-balloon) 구조

컨텐츠는 버튼 하단에 맞추어 자동으로 조정

버튼의 속성값 중 data-top / data-left 값을 추가하면 컨텐츠 위치 조정됨.

말풍선 위쪽 화살표는 after 요소에 border 로 생성. 디자인 변경시 CSS 수정 및 제거.

말풍선 내용 입니다. 
<button type="button" class="btn-balloon btn-s btn-white" data-top="10" data-left="0"><span>말풍선 띄우기</span></button>
<div class="cnt-balloon">
	<div class="cnt">
		말풍선 내용 입니다.
	</div>
	<button type="button" class="btn-balloon-close">말풍선 닫기</button>
</div>
