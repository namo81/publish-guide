컨텐츠 포함

컨텐츠와 묶어서 사용할 경우

탭메뉴 버튼의 href 값을 통해 해당 컨텐츠 hide / show

탭메뉴의 갯수와 컨텐츠 갯수는 일치시켜야 함.

  • 탭메뉴 1
  • 탭메뉴 2
  • 탭메뉴 3
탭 컨텐츠 111
탭 컨텐츠 222
탭 컨텐츠 333 
<div class="tab-wrap ex1">
	<div class="tab-menu tab-type1 count3">
		<ul>
			<li class="on"><a href="#tab-cnt1">탭메뉴 1</a></li>
			<li><a href="#tab-cnt2">탭메뉴 2</a></li>
			<li><a href="#tab-cnt3">탭메뉴 3</a></li>
		</ul>
	</div>

	<!-- 탭 컨텐츠 1 -->
	<div class="tab-cnt" id="tab-cnt1">
		탭 컨텐츠 111
	</div>

	<!-- 탭 컨텐츠 2 -->
	<div class="tab-cnt" id="tab-cnt2">
		탭 컨텐츠 222
	</div>

	<!-- 탭 컨텐츠 3 -->
	<div class="tab-cnt" id="tab-cnt3">
		탭 컨텐츠 333
	</div>
</div>

JS 설정

1개 단독 설정

var 변수명 = new nTabMenu({
	wrap         : '.tab-wrap', - 탭 메뉴 및 컨텐츠를 포함하는 영역 선택자
	menuWrap     : '.tab-menu', - 탭 메뉴 선택자
	tabCnt       : '.tab-cnt'   - 탭 컨텐츠 선택자
 });

동일구조 다수 tab 전체 적용

버튼 영역 클래스는 'tab-menu'

컨텐츠 영역 클래스는 'tab-cnt'

탭 안의 탭 같은 depth 구조에서는 사용불가

nTab('css 선택자'); - 탭 메뉴 및 컨텐츠를 포함하는 영역 선택자
