	<label for="" class="hide">기본형 텍스트</label>

		<span class="tx-red">내용을 입력해주세요.</span>
	</div>
</div>


// 단독요소 설정

radio 버튼

일반형 (radio + label)

label 의 for 값과 input radio 의 id 값을 일치 시켜주어야 함.

기본적으로 연속 2개를 사용할 경우 2번째 부터 margin-left:15px 가 자동으로 적용

.box 의 위치에 따라 checkbox 아이콘 위치 이동 // radio 만 보여야 할 경우 .box 안에 텍스트 입력

가장 외곽 div 의 min-height 는 radio 아이콘 사이즈보다 2px 크게 설정

<div class="inp-radio">
	<input type="radio" name="radio-set" id="no" checked>
	<label for="no"><span class="box"></span> 라디오버튼 - N</label>
</div>
<div class="inp-radio">
	<input type="radio" name="radio-set" id="yes">
	<label for="yes">라디오버튼 - Y <span class="box"></span></label>
</div>
<div class="inp-radio">
	<input type="radio" name="radio-set" id="non-tx">
	<label for="non-tx"><span class="box">라디오버튼 - Y</span></label>
</div>

가장 외곽 div 의 min-height 는 checkbox 아이콘 사이즈보다 2px 크게 설정

	<select name="" title="">
		<option value="">직접 선택</option>
		<option value="">nate.com</option>
		<option value="">daum.net</option>
	</select>
</div>
