호출버튼

스타일은 버튼 관련 css 에서 별도 설정

기능용 클래스 지정 후 script 에서 설정 (ex : open-layer 클래스)

data-target 에 호출할 레이어 팝업 ID 기입

동일 클래스가 다수일 경우 공통 적용

<button type="button" data-target="layer1" class="btn small open-layer">레이어팝업1</button>
	
nlayer.showBtn('호출버튼 클래스'); // 버튼 클릭 호출형
nlayer.showFunc('호출할 레이어 ID'); // 함수형 호출 - 대상 팝업 id 값 설정

레이어팝업 컨텐츠

디테일한 스타일은 css 로 설정

'layer-mid' 는 기본적으로 화면 최대사이즈 기준으로 내부 scroll 자동생성

내부 scroll 미사용 시 layer-mid 클래스 제거

닫기버튼 클래스 'close-layer' 사용

show 된 레이어가 여러개일 경우 - 모두 닫고자 할 때 close-layer 에 'all' 클래스 추가

<div class="layer" id="layer2">
	<div class="inbox"> <!-- in-scroll 추가 시 내부 스크롤 -->
		<div class="layer-cnt">
			<div class="layer-top">test</div>
			<div class="layer-mid">
				<button type="button" class="close-layer all">전체닫기</button>
				<button type="button" class="close-layer">닫기</button>
			</div>
		</div>
	</div>
</div>
