CSS

/* Toast 및 Snack Bar =============================================================== */
.toast { position:fixed; display:none; width:100%; height:100%; top:0; left:0; z-index:1001; }
.toast .toast-cnt { position:relative; width:90%; top:5%; left:5%; padding:15px; background-color:#fff; border-radius:5px; box-sizing:border-box; box-shadow:1px 1px 6px rgba(0,0,0,.3); transform:translateY(-1000px); }
.toast.show { display:block; }
.toast.show .toast-cnt { transform:translateY(0); animation:toastShow 3s; }

.toast.bottom .toast-cnt { top:95%; transform:translateY(-100%); }

Toast Bar

스타일 및 show 모션은 css 에서 별도 설정

nToast 함수에 문구를 파라미터로 전달하여 실행

CSS 에서 animation 시간 설정에 따라 전체 시간 조정됨
전체 시간에서 show 모션 20% / show 상태 유지 60% / hide 모션 20% 로 설정되어 있음
ex) 전체 5초일 경우 show 모션 1초 / show 상태유지 3초 / hide모션 1초

CSS 에서 toastShow 부분 조정 시 각 시간 조정 가능

<button type="button" class="btn medium test">토스트 테스트</button>

var btn = document.querySelector('.btn.test');
	
btn.addEventListener('click', function(){
	nToast('토스트 테스트 중 입니다. 음하하하');
});

Toast Bar - 클래스 추가

nToast 함수에 2번째 파라미터로 클래스 전달 시 toast 팝업에 클래스 추가 가능

ex) nToast('문구', 'bottom') 설정 시 <div class="toast bottom"> 으로 설정

** bottom 추가 시 toast 가 하단에서 표출됨

그 외 기타 class 추가 후 CSS 로 스타일 제어 가능

var btn2 = document.querySelector('.btn.test2');
	
btn2.addEventListener('click', function(){
	nToast('토스트 테스트 중 입니다. 음하하하', 'bottom');
});
