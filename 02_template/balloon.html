클릭형

기본형에 'click' 클래스 추가 ( hover 기능 작동 X )

balloon 에 추가클래스 (top / center / left / right) 추가로 위치 변경 가능
(* center 와 top 은 중복사용 가능)

btn-bln 클릭 시 'balloon-wrap 에 'on' 클래스 토글 기능이 common.js 에 포함되어 있음. (nBalloonTgl 함수)

btn-bln-close 버튼은 닫기 기능용 - 추가옵션으로 없어도 무관함

말풍선 내용이 들어가는 영역입니다. 
<div class="balloon-wrap click">
	<button type="button" class="btn-bln"><span>버튼</span></button>
	<div class="balloon">
		<div class="cnt">
			말풍선 내용이 들어가는 영역입니다.
		</div>
		<button type="button" class="btn-bln-close">말풍선 닫기</button>
	</div>
</div>
