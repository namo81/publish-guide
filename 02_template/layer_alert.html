알럿 / 컨펌

알럿

기본 스타일은 레이어팝업과 공유하며, css 에서 별도 설정 가능

알럿 관련 클래스 추가 후 javascript 에서 기능 설정 (ex : alert 등..)

알럿 함수 파라미터 : nlayerAlert('내용', '타이틀', 확인 후 실행할 함수, '버튼 텍스트');

첫번째 '내용' 텍스트는 필수항목임

실제 팝업 관련 html 은 자동생성되므로 별도 하드코딩 X

팝업 style은 css 로 설정 / 특정 클래스 필요 시 layerpop-v4.0.0.js 파일 내 항목 수정

<button type="button" class="btn01 alert">알럿 호출</button>

var btnAlert = document.querySelector('.alert');
btnAlert.addEventListener('click', function(){
nlayerAlert('내용을 확인하셨습니까?', '타이틀', function(){ console.log('check!!') });
});

컨펌

기본 스타일은 레이어팝업과 공유하며, css 에서 별도 설정 가능

컨펌 관련 클래스 추가 후 javascript 에서 기능 설정 (ex : confirm 등..)

컨펌 함수 파라미터 : nlayerAlert('내용', '타이틀', '확인 시 실행할 함수, '취소버튼 텍스트', '확인버튼 텍스트');

첫번째 '내용' 텍스트는 필수항목임

'확인' 시 실행할 함수는 별도 생성 후 함수명 기입

실제 팝업 관련 html 은 자동생성되므로 별도 하드코딩 X

팝업 style은 css 로 설정 / 특정 클래스 필요 시 layerpop-v4.0.0.js 파일 내 항목 수정

<button type="button" class="btn01 confirm">컨펌 호출</button>

var btnConfirm = document.querySelector('.confirm');
btnConfirm.addEventListener('click', function(){
var funcRun = function(){
	console.log('YES');
}
nlayerConfirm('내용을 확인하셨습니까?', '타이틀', funcRun)
});
