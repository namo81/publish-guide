SVG 가이드

svg 기본구조

svg 파일 별도 생성

<?xml version="1.0" encoding="utf-8"?>
<!DOCTYPE svg PUBLIC "-//W3C//DTD SVG 1.1//EN" "http://www.w3.org/Graphics/SVG/1.1/DTD/svg11.dtd">
<svg version="1.1" id="Layer_2" xmlns="http://www.w3.org/2000/svg" xmlns:xlink="http://www.w3.org/1999/xlink" x="0px" y="0px" viewBox="0 0 100 100">
	<!-- svg 요소 추가 -->
</svg>

inline 으로 직접 넣을 경우

<svg width="100" height="100">
	<!-- svg 요소 추가 -->
</svg>

단순 이미지 사용

단순 이미지 형식으로 사용할 경우 img 태그 / background 설정 등으로 사용 가능

기본적인 width / height 설정 가능

벡터 형식이므로, 이미지 깨짐 현상 없음 - 모바일 / 웹 모두 1개 파일로 사용 가능

img 태그

<img src="svg/icon_arrow-type1.svg">

background 설정

<div class="svg-box></div>
.svg-box { width:50px; height:50px; background:url(svg/icon_arrow-type1.svg) no-repeat left top; }

inline 사용

html 파일 내 직접 코드 작성

일반적인 html 태그처럼 사용 가능하며, CSS 로 조작이 자유로움

단, 그려진 요소가 svg 의 width / height 에 대응하지 않는 고정값이므로, 크기 조정 등의 작업이 번거로움

<svg width="100" height="100">
	<polyline fill-rule="evenodd" clip-rule="evenodd" fill="none" stroke="#000000" stroke-width="10" stroke-linecap="round" stroke-linejoin="round" stroke-miterlimit="10" points="32.279,12.497 69.958,50.174 32.258,87.873 "/>
</svg>

Object 사용

css 로 svg 요소에 대한 추가제어가 필요할 경우 사용 (color 등)

단, svg 용 CSS 를 따로 만들어야 하며, 해당 svg 파일에서 css 파일을 로드하여야 함

<object data="svg/icon_arrow-type1.svg" type="image/svg+xml"></object>
<?xml version="1.0" encoding="utf-8"?>
<!DOCTYPE svg PUBLIC "-//W3C//DTD SVG 1.1//EN" "http://www.w3.org/Graphics/SVG/1.1/DTD/svg11.dtd">
<?xml-stylesheet type="text/css" href="style.css"?>
<svg version="1.1" id="Layer_2" xmlns="http://www.w3.org/2000/svg" xmlns:xlink="http://www.w3.org/1999/xlink" x="0px" y="0px" viewBox="0 0 100 100" enable-background="new 0 0 100 100" xml:space="preserve">
	<polyline fill-rule="evenodd" clip-rule="evenodd" fill="none" stroke="#000000" stroke-width="10" stroke-linecap="round" stroke-linejoin="round" stroke-miterlimit="10" points="32.279,12.497 69.958,50.174 32.258,87.873 "/>
</svg>

기타

단순 이미지 사용은 괜찮으나, 모션이 들어갈 경우 IE 에서 작동되지 않음.
