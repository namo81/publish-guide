Nanum Gothic

Google Webfont Helper

파일 출처 : 구글 웹폰트 헬퍼

폰트 두께 : 400 / 700 / 800

폰트 파일 타입 : woff / woff2 / ttf / eot / svg

하단 CSS는 참고용 기본 두께(regular) CSS 이며, 각 두께별 설정은 css 파일 내 모두 선언되어 있음.

모든 브라우저(Full)

구글-나눔고딕_full_down (css + font 파일)

@font-face {
	font-family: 'Nanum Gothic';
	font-style: normal;
	font-weight: 400;
	src: url('nanum-gothic-v17-korean-regular.eot'); /* IE9 Compat Modes */
	src: local(''),
		url('nanum-gothic-v17-korean-regular.eot?#iefix') format('embedded-opentype'), /* IE6-IE8 */
		url('nanum-gothic-v17-korean-regular.woff2') format('woff2'), /* Super Modern Browsers */
		url('nanum-gothic-v17-korean-regular.woff') format('woff'), /* Modern Browsers */
		url('nanum-gothic-v17-korean-regular.ttf') format('truetype'), /* Safari, Android, iOS */
		url('nanum-gothic-v17-korean-regular.svg#NanumGothic') format('svg'); /* Legacy iOS */
}

최신 브라우저(Modern)

구글-나눔고딕_modern_down (css + font 파일)

@font-face {
	font-family: 'Nanum Gothic';
	font-style: normal;
	font-weight: 400;
	src: local(''),
			url('nanum-gothic-v17-korean-regular.woff2') format('woff2'), /* Chrome 26+, Opera 23+, Firefox 39+ */
			url('nanum-gothic-v17-korean-regular.woff') format('woff'); /* Chrome 6+, Firefox 3.6+, IE 9+, Safari 5.1+ */
}

Naver 나눔폰트

파일 출처 : 네이버 나눔폰트

폰트 두께 : Light(200) / Regular(400) / Bold(700) / Extra Bold(900)

폰트 파일 타입 : woff / ttf

Regular 두께만 eot 타입파일 추가

하단 CSS는 참고용 기본 두께(regular) CSS 이며, 각 두께별 설정은 css 파일 내 모두 선언되어 있음.

모든 브라우저(Full)

네이버-나눔고딕_full_down (css + font 파일)

@font-face {
	font-family: 'NanumGothic';
	font-style: normal;
	font-weight: 400;
	src: url(NanumGothic.eot); /* regular 만 있음. */
	src: local(''), 
		url(NanumGothic.woff) format("woff"), 
		url(NanumGothic.ttf) format("truetype");
}

최신 브라우저(Modern)

네이버-나눔고딕_modern_down (css + font 파일)

@font-face {
	font-family: 'NanumGothic';
	font-style: normal;
	font-weight: 400;
	src: local(''), 
		url(NanumGothic.woff) format("woff");
}
