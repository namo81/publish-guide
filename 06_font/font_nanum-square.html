Nanum Square

기본

파일 출처 : 네이버 나눔폰트

폰트 두께 : Light(200) / Regular(400) / Bold(700) / Extra Bold(900)

폰트 파일 타입 : woff / ttf / eot

하단 CSS는 참고용 기본 두께(regular) CSS 이며, 각 두께별 설정은 css 파일 내 모두 선언되어 있음.

모든 브라우저(Full)

나눔스퀘어_full_down (css + font 파일)

@font-face {
	font-family: 'NanumSquare';
	font-style: normal;
	font-weight: 400;
	src: url(NanumSquareR.eot);
	src: local(''), 
		url(NanumSquareR.eot?#iefix) format("embedded-opentype"), 
		url(NanumSquareR.woff) format("woff"), 
		url(NanumSquareR.ttf) format("truetype");
}

최신 브라우저(Modern)

나눔스퀘어_modern_down (css + font 파일)

@font-face {
	font-family: 'NanumSquare';
	font-style: normal;
	font-weight: 400;
	src: local(''), 
		url(NanumSquareR.woff) format("woff");
}
