Noto Sans Kr

Google Webfont Helper

파일 출처 : 구글 웹폰트 헬퍼

폰트 두께 : 100 / 300 / 400(Regular) / 700 / 800

폰트 파일 타입 : woff / woff2 / eot / svg

** noto sans kr 의 경우 구글에 css 는 아래와 같으나, 실제 ttf 파일은 누락된 상태

하단 CSS는 참고용 기본 두께(regular) CSS 이며, 각 두께별 설정은 css 파일 내 모두 선언되어 있음.

모든 브라우저(Full)

NotnSansKr_full_down (css + font 파일)

@font-face {
	font-family: 'Noto Sans KR';
	font-style: normal;
	font-weight: 400;
	src: url('noto-sans-kr-v21-korean-regular.eot'); /* IE9 Compat Modes */
	src: local(''),
			url('noto-sans-kr-v21-korean-regular.eot?#iefix') format('embedded-opentype'), /* IE6-IE8 */
			url('noto-sans-kr-v21-korean-regular.woff2') format('woff2'), /* Super Modern Browsers */
			url('noto-sans-kr-v21-korean-regular.woff') format('woff'), /* Modern Browsers */
			url('noto-sans-kr-v21-korean-regular.ttf') format('truetype'), /* Safari, Android, iOS */
			url('noto-sans-kr-v21-korean-regular.svg#NotoSansKR') format('svg'); /* Legacy iOS */
}

최신 브라우저(Modern)

NotnSansKr_modern_down (css + font 파일)

@font-face {
	font-family: 'Noto Sans KR';
	font-style: normal;
	font-weight: 400;
	src: local(''),
			url('noto-sans-kr-v21-korean-regular.woff2') format('woff2'), /* Chrome 26+, Opera 23+, Firefox 39+ */
			url('noto-sans-kr-v21-korean-regular.woff') format('woff'); /* Chrome 6+, Firefox 3.6+, IE 9+, Safari 5.1+ */
}
