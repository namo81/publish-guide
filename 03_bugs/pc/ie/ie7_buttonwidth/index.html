button 태그 width 버그

상세 현상

web 화면 작업 시 라운드된 버튼 때문에 좌측/우측 이미지 2종으로 버튼을 만들 경우.

IE 7 에서 a 태그에서는 크게 문제가 없으나, button 태그를 고려하여 작업 할 경우 button 태그의 width 값이 그 안의 span 값과는 무관하게 버그성으로 커지는 현상이 있음.

해결 방법

button 태그에 overflow:visible 설정을 해주면 넓이값이 span 와 같이 맞추어짐.

상세 현상

button 안의 span 태그가 텍스트 길이를 인식하지 못하고 글자 1개만 노출되는 버그도 발생

해결 방법

span 태그에 white-space:nowrap 설정 추가
