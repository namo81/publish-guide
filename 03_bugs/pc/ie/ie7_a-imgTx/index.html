A 태그 내 이미지+시스템 폰트 구성 클릭버그

상세 현상

A 태그 내에 이미지 + 시스템 폰트로 구성되어있을 경우

IE 7 에서 A 태그 내의 이미지 영역에 클릭 event 가 실행되지 않는 버그. - 시스템폰트 영역은 정상 작동.

해결 방법

A 태그 내 이미지 및 이미지를 감싸는 영역에 width,height 등의 사이즈를 특정하는 CSS 속성을 제거.

해결 방법

width,height 등 사이즈 속성을 사용하여야 할 경우 a 태그 마지막에 클릭용 투명 영역를 넣어서 A 태그 전체 사이즈 만큼 최상단에 배치하여 사용.
