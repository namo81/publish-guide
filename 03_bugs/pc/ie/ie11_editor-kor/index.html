Editor 등에서 inline-block 영역에 한글 입력 시 '자음' 2중입력 현상

상세 현상

CKEditor / summernote 같은 html Editor 입력 플러그인을 사용할 때, 입력폼이 display:inline-block 요소 하위일 경우 라인마다 최초 입력 시 한글 자음이 2번 들어가는 현상

editor 에서 ul / ol 등의 태그로 문구 서식 추가할 경우 해당 태그에 display가 inline-block 일 때 발생하였음.

해결 방법

IE11 만의 독특한 버그로 해당 inline-block 요소를 block 로 변경 시 해결 됨.
