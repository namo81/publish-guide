translate 모션 이동시 반짝이는 현상

상세 현상

translateX ( or translateY ) 를 사용하여 배너나 페이지 이동시 모션 종료 후 webview 화면에서 번쩍이는 현상이 발생.

Y 및 Z 값 설정이 없이 X 만 움직일 경우 (Y만 움직이는 경우도 동일할듯) 모션 종료 시 설정되지 않은 값의 갱신이 발생하는 듯.

해결 방법

이 부분은 translate3d(0,0,0) 으로 고쳐주고 원하는 x,y 값을 지정해주면 문제 해결.

해당 부분은 모션 움직임의 퍼포먼스 향상에도 효과가 있음.

모든 이동은 translate3d(0,0,0) 을 사용.
