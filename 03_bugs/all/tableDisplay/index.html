테이블 toggle 구현 시 display 관련

상세 현상

테이블 자체에 toggle 기능을 display 속성을 통해 구현할 경우

td 및 tr 에 display 설정을 하게 되면 table 자체 레이아웃이 깨지는 현상이 발생.

display:table-row 로 설정하면 ie7 에서 먹히지 않음.

해결 방법

CSS 에 display 설정을 하지 않고, hide 될 때는 display:none / 다시 show 시킬 때는 display: 으로 설정.

아래 예시 참고

$('클래스').css('display','none');
$('클래스').click(function(){
	$(this).css('display','');
});

Object.style.display = 'none';
Object.style.display = '';
