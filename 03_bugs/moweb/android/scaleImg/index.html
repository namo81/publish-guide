transform scale 적용 시 이미지 깨짐 현상

상세 현상

스케일을 이용한 애니메이션에서 사이즈가 작은 상태에서 크게 키워지는 모션을 사용할 때 이미지가 깨지는 현상.

이미지를 감싸는 태그에 애니메이션을 주고, 이미지는 width:100% 을 한 상태일때.
이미지를 감싸는 태그에 애니메이션을 주게 되면, 태그의 scale 이 0 일때의 width 값을 이미지에 적용하고 그대로 유지하면서 커지는 버그가 발생. (매우 작은 이미지를 강제로 늘인듯 한 느낌)

특정 안드로이드 버전 및 특정 폰에서 발생하는 버그

해결 방법

img 태그에 직접 애니메이션을 적용하면 해결됨.
