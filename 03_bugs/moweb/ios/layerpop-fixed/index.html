레이어팝업 띄울 때 fixed 요소 위치 오류

상세 현상

T적금 그룹 프로젝트 진행 시 발생한 오류 / 같은 OS 인데도 아이폰 6 이상에서만 발생.

화면 상단에 fixed 타이틀 영역이 있는 페이지에서 레이어 팝업을 띄워야 하는 상황
레이어 팝업 스크립트가 기본 페이지의 스크롤을 막기 위해서 기본 페이지의 position 값을 fixed 로 변경하도록 설정되어 있었는데,
기본 페이지의 position 값이 fixed 로 변경되면서 원래 fixed 였던 요소의 위치가 화면 중앙으로 튀는 현상이 발생.

해결 방법

레이어 팝업 띄울 시 기본 페이지의 스크롤 방지를 fixed 가 아니라,
스크립트로 높이값 계산 후 height 값 설정하고, overflow:hidden 설정하여 진행.

var winH = $(window).innerHeight();
$('기본페이지').css({'overflow':'hidden', 'height':''+winH+'px'});
