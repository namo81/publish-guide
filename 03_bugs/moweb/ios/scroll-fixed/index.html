스크롤 관련 이벤트

상세 현상

gnb 메뉴가 스크롤 되다가 상단에 붙는 기능 구현 시, gnb 를 2개로 놓고, fixed 용 gnb 를 hide 시켰다가 show 할 경우

아이폰 사파리에서 스크롤이 다 끝난 후 show 되는 현상. - ios 의 스크롤 event 전달 시점 문제?

해결 방법

fixed 요소를 hide / show 가 이나라, 클래스 추가를 통해 transform:translate3d 이동으로 숨겼다가 보여주는 방식으로 구현하면 정상 작동함.

클래스 추가를 하더라도, display:none - block 설정을 사용하거나, top:-100px 등의 top 속성으론 안됨.

transform 은 이미 그려진 요소를 이동 시키는 방식 / 그외 display 제어나 top 의 경우 요소를 새로 그리는 방식이라 안되는 듯 함.
